# Security Policy

## Supported Versions

We currently support security fixes for the latest release on the `main` branch.

## Reporting a Vulnerability

Please report security issues via GitHub Security Advisories:

1. Go to the repository's **Security** tab.
2. Click **Report a vulnerability**.
3. Provide a clear description, steps to reproduce, and potential impact.

We aim to acknowledge reports within 72 hours and provide a remediation plan as quickly as possible.
